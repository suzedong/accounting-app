# Prompt 文件

AI 记账助手的 SYSTEM_PROMPT 存储在这些文件中，每次请求时由 `server.py` 动态读取。

## 文件说明

| 文件 | 用途 | 端点 |
|---|---|---|
| `dispatch.md` | 意图识别 + 参数提取（多 Skill 路由） | `POST /api/ai/dispatch` |
| `record.md` | 纯记账解析（降级方案） | `POST /api/ai/parse` |

## 编写规范

1. **Markdown 格式**：使用 `#` 标题、`##` 二级标题等结构化标记，便于 agent 理解
2. **分类关键词独立成段**：每个分类一个 `####` 标题，关键词用顿号分隔，便于 agent 追加
3. **规则编号**：重要规则用编号列表，便于 agent 引用和修改
4. **不要删除已有规则**：agent 追加关键词时应保留原有内容
5. **修改后无需重启服务**：server.py 每次请求重新读取文件

## Agent 自修改方式

Agent 可以通过自然语言对话修改 prompt：
- "把 '瑞幸' 加到餐饮分类关键词里"
- "中移金科应该分到通信费，更新分类规则"
- "银行通知的金额识别需要支持更多格式"

修改后的内容会直接写入对应文件，立即生效。
